HTML

Hypertext Markup Language (HTML) is the standard Markup language for creating web pages and web applications. Web brouwsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appeareance of the document.
